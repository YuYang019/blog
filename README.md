# blog
写的一个简单的博客，前后端分离

前端：vue + vuex + vue-router + axios + elementUi
后端：koa2 + mongodb + mongoose
登录认证: jwttoken

后端不是很会，参考的项目 https://github.com/jackhutu/jackblog-api-koa
